Make all the naked URLs into a live links

== Description ==

<strong>Naked URLs to live links Plugin contains:</strong> -
<ul>
<li>The plugin's fuctionality is enabled by adding a filter to the content</li>
<li>The plugin does damage if the post(page) content has embeded code or Javascript</li>
<li>For this reason, the plugin checks for two words - 'iframe' and 'text/javascript'. If the post contains any of them, the plugin functionality won't be applied to this post(page).</li>
<li>Supported characters in the URL addresses: */.?_^%$*#!~+- </li>
<li>The comments in the plugin contains detailed explanation how the code works. That makes adding additional functionality easy for the webmasters.</li>

</ul>
<strong>Known issues:</strong>
<ul>
<li>If any of the naked URL address is a link to a video, this will disable the plugin for this post(page). 
Examples: Ted videos, YouTube, Vimeo clips
</li>
<li>If you have embedded code for YouTube & Vimeo clips(via iframe or via a media embedding Wordpress plugin), that won't affect <strong>Naked URLs to live links</strong> plugin.
</li>
</ul>

== Installation ==

1. Upload `nakedurl.zip` onto your local computer.
2. Go to your WordPress Dashboard and select <strong>Plugins >> Add New</strong>.
3. Click on the <strong>Upload</strong> option at the top and select the `nakedurl.zip file` you just downloaded.
4. Click on <strong>Install</strong>.
5. Activate the plugin through the 'Plugins' menu in WordPress
